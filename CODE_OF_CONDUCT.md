# Code of Conduct

## Our Standard

Please keep all collaboration respectful, inclusive, and constructive.

## Expected Behavior

- Be respectful in communication and code review.
- Assume good intent and focus on technical outcomes.
- Give actionable feedback grounded in facts.

## Unacceptable Behavior

- Harassment, discrimination, or abusive language.
- Personal attacks or intentionally disruptive behavior.

## Scope

This applies to repository discussions, pull requests, issues, and other project spaces.

## Reporting

If you experience unacceptable behavior, open a private maintainer contact through repository admins.
